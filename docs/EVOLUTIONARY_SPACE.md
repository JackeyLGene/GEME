# 达尔文主义进化空间（刘氏框架）

> 2026-05-14 22:43 确立。

## 三个维度

### 维度一：生存（从被动接受到趋利避害）

| 阶段 | GEME 对应 | 描述 |
|------|-----------|------|
| L1 被动接受 | 输入→竞争合并 | 细度只能接收环境中的营养分子，没有选择能力 |
| L2-L3 模式识别 | 共现→桥接 | 细度开始"知道"哪些环境信号经常一起出现 |
| L4-L5 主动预测 | L4 预测层 | 不再等营养来——主动向高预测区移动 |
| L6 趋利避害 | sys_doubt / 确信 | 判断："这边好，那边不好"——移动由判断驱动 |

### 维度二：交流（从被动释放信息素到叫声到语言）

| 阶段 | GEME 对应 | 描述 |
|------|-----------|------|
| 被动释放 | L6 单向输出 | 细度释放 AHL 分子——不是"有意交流"，只是代谢副产物 |
| 信号交换 | 邻居向量交换 | 一个细度的 L6 进入另一个细度的帧经济——交流发生 |
| 符号系统 | 桥签名 σ | 信号有了格式（──，◯◯，pred_）——交流有了语法 |
| 语言 | σ 的可复制性 | 信号可以被另一个系统解读、复制、修改——语言诞生 |

### 维度三：外置（从被动的信息场到外化的工具到知识）

| 阶段 | GEME 对应 | 描述 |
|------|-----------|------|
| 被动信息场 | 共享环境向量 | 细度的信号分子扩散到环境中——信息场形成 |
| 环境记忆 | 信息场的累积 | 分子浓度梯度保留了过去行为的痕迹——"环境记住了" |
| 外化工具 | MIDI / 乐谱 | 人类把节律刻在石头上、写在纸上——"思想外置" |
| 知识独立演化 | EE 的自反馈 | 外置思想有了自己的演化动力——不再依赖创造者 |

## 三个维度的关系

```
生存（垂直维度）：
  个体在时间中的自指深化——"我"的复杂度

交流（水平维度）：
  个体之间的信号交换——"我们"的连通度

外置（深度维度）：
  信号脱离个体的持续存在——"它"的独立演化

  任何生命/认知现象都可以在这三个维度中定位：

    细菌：生存 L2（被动识别模式）
          交流 L1（被动释放信息素）
          外置 L1（信息场微弱、瞬间消散）

    人类个体：生存 L6（趋利避害决策）
             交流 L5（语言）
             外置 L3（工具使用但不创造知识）

    人类文明：生存 L6
              交流 L6
              外置 L6（图书馆、互联网、AI）
```

## 对应三部曲

- **第一篇 GEME**: 定义了维度一（生存）的基元——三个常数驱动六个层级的自指深化
- **第二篇 BGM**: 展示了维度二+维度三的萌芽——多 GEME 的信息场和节律同步
- **第三篇 EE**: 完成维度三——外置引擎的反向驱动机制

> 这不是三个独立的论文主题——这是一个三维坐标系中的三个投影。
