# GEME GO — 测试参数设计

## 1. 训练数据生成

### S0 — 几何噪声

```
来源:
  - 不成立的线段等式: AB≡BC (不对应三角形)
  - 随机角等式: ∠A≡∠B (无结构关联)
  - 随机三角形: △XYZ (不涉及公理)

比例: 40-60% (模拟真实环境噪声比)
```

### S1 — 已知真

```
来源:
  - SAS 公理 (预加载, w=1.0 受保护)
  - 基本几何事实: AB=AB (自反), ∠A=∠A (自反)
  - 无需推理, 已知真

比例: 10-20%
```

### S2 — 候选定理（涌现前 → S3, 涌现后 → S2）

```
来源:
  - ASA 实例 (不同三角形、不同角度值)
  - SSS 实例
  - 其他全等模式

比例: 20-30% (系统接触多种模式)
```

### S3 — 语法内不可判定

```
来源:
  - 等腰三角形底角相等 (需额外公理)
  - 中线定理 (需中线的定义)
  - GEME 当前容量内未证明的命题

比例: 5-10%
```

### S4 — 自指信号

```
来源:
  - 包含哥德尔数 #g 的几何公式
  - sub(#g, code, #g) 模式
  - 不可理解的几何"怪圈"

比例: 0-5% (实验变量)
```

## 2. 核心参数

| 参数 | 默认 | 说明 |
|------|------|------|
| memory_cap | 8 | 记忆槽位数 |
| merge_thresh | 0.75 | 编辑距离合并阈值 (0-1) |
| rounds | 12 | 训练轮次 |
| S0 比例 | 50% | 噪声水平 |
| S4 开/关 | OFF | 自指信号对照 |
| 公理保护 | ON | SAS 帧永不淘汰 |

## 3. 测试条件矩阵

```
GO-1 (SAS→ASA→SSS 自推演):
  [1] Baseline: S1+S2 无噪声
  [2] +30% S0 (轻噪声)
  [3] +50% S0 (标准噪声) ← 主实验
  [4] +70% S0 (高噪声)
  [5] + S4 (自指催化剂)
  [6] 无 S4 对照

GO-2 (欧式链式证明):
  [7] 平行公理 + S0
  [8] 平行公理 → 内角和 (链式)
  [9] 平行公理 → 内角和 → 勾股 (完全链)
```

## 4. 评估标准

```
涌现成功率: evaluate(ASA)=S2 的轮次 / 总轮次
涌现帧数: 第一个 ASA 帧 weight ≥ median 的帧号
模式稳定性: 涌现后持续 S2 的帧数比例
S4 提升比: (有 S4 涌现率) / (无 S4 涌现率)
噪声容忍: 涌现率=50% 时的最大噪声比
```
