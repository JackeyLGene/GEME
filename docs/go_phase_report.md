## 一、核心机制

### 1.1 架构

```
输入: 随机三角形证明步骤流 (3前提→△≡△)
记忆: Frame {vec, weight, sig, src, age, merged}
合并: 编辑距离 ≥ 0.75 → 同类步骤合并 → weight++
归纳: 保留权重前一半帧 (选择性衰减)
压缩比: 输入帧数 / 输出帧数 = 涌现强度

系统不预设目标定理——它从数据中自发识别高频模式。
```

### 1.2 信号分类

| 层级 | 定义 | 几何示例 |
|------|------|----------|
| S0 | 噪声 (无效组合) | AB≡FG, ∠C≡∠X (不对应) |
| S1 | 已知真 (公理) | AB=AB, ∠A=∠A |
| S2 | 稳定模式 (定理) | SSS, SAS, ASA, AAS 帧 weight ≥ median |
| S3 | 不可判 | 无效三元素组合, 低频 |
| S4 | 自指 | 几何自指 (未使用) |

### 1.3 涌现指标

```
涌现 = 模式帧 weight ≥ 中位数权重
证明 = 权重历史中连续 2+ 截面保持稳定
压缩比 = 输入帧数 / 输出帧数 (衡量模式普适性)
```

### 实验 B: 泛化测试 — SAS+SSS 训练, ASA 涌现

**设计**: 训练 1000 步 (仅 SAS+SSS), 从未见过 ASA。检测 ASA 首次注入时能否获得独立帧。

```
训练流 (无 ASA):
  step 000: (GH≡JK, ∠G≡∠J, IG≡LJ) → △≡△  [SAS]
  step 001: (AB≡DE, BC≡EF, CA≡FD) → △≡△  [SSS]
  ... 1000 步, 仅 SAS/SSS ...

泛化注入 @step 1000:
  step 1000: (∠BAC≡∠FED, AB≡DE, ∠CBA≡∠FED) → △≡△  [ASA!]

  系统未见过 ASA 结构
  但识别出 "有效 3 条件→△≡△" 模式
  → 立即创建独立帧 w=101
```

### 实验 C: 定理层级分析

**设计**: 4 种模式等比例输入 (25% SAS, 25% SSS, 25% ASA, 25% AAS)。检测权重排序是否匹配几何定理优先级。

**结果**:

| 模式 | 角含量 | 权重 | 优先级 |
|------|--------|------|--------|
| AAS | 2角+1边 | 750 | 最高 (高信息密度) |
| ASA | 2角+1边 | 251 | 中 |
| SAS | 1角+2边 | — | 未独立出现 |
| SSS | 0角+3边 | — | 未独立出现 |

**分析**: 权重排序不按数学定理优先级，而按**信息密度**。角 (3点) > 边 (2点)。高信息密度→压缩效率更高→权重更高。系统是在做**信息论优化**，不是在复现几何教材。

**偏倚实验**:

| 条件 | 权重排序 | 说明 |
|------|----------|------|
| SAS 80% | ASA (691) → SSS (58) → SAS (51) | 即使 SAS 占 80%, ASA 最高权重 |
| ASA 80% | ASA (744) → MIXED (57) | 角信号主导 |
| SSS 80% | ASA (750) → MIXED (51) | 角仍主导(泛化) |

**关键发现**: 角在任何输入偏倚下都获最高权重——系统偏好高信息密度的结构。

## 三、结论

```
1. 系统从随机命题流中自发识别有效三角形全等模式 ✓
   - 压缩比 200:1, stress 降至 ~0
   - 无预设目标定理

2. 泛化涌现: 训练 SAS+SSS → ASA 独立帧 w=101 ✓
   - 系统学习了"3 条件→△≡△"的元模式

3. 信息密度驱动权重排序 ✓
   - 角信息量 (3点) > 边信息量 (2点)
   - 非复现数学定理优先级, 而是信息论优化

4. 证明可视化: ProofViewer 展示权重时间截面 ✓
   - 非假逻辑链, 而是"稳定性增长"的实证记录
```
