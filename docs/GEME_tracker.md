**Started:** 2026-05-03 · **Status:** Active

## Mission

GEME is a cognitive oscilloscope — an observation instrument that makes
structural emergence visible to the human observer.

GEME does exactly three things:
1. **Readability Mapping** — signature generation, domain assignment
2. **Diagonal Transforms** — self-reference, Godel construction
3. **Training Evaluation** — boundary detection, harm, emergence

GEME is not an agent. Pattern recognition belongs to the human.

---

## Architecture

```
Human → Wittgenstein Table → Digital → GEME (①②③) → Emergence
                                                    →
Layer 2 GEME (algebraic interpretation) → Theorem → Human
```

Core innovation: **Recursive self-observation** — Layer 2 GEME interprets
Layer 1 GEME's structural clusters as provable theorems.
