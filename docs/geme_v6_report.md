## 一、核心架构

```
输入 → W-table 11 维符号频次向量
     → 记忆队列: (v, weight), 同类合并 (weight += 1)
     → stress = utilization × (1 - efficiency) ∈ [0,1]
     → stress > 0.6 → 归纳: 保留 weight ≥ median 的帧
     → evaluate: 
          模式帧 weight ≥ median ∧ 编辑距离匹配 ≥ 动态阈值 → S2
          分类器可证明 → S1/S2; 否则 → S3
```

**无 InductionRule, 无梯度, 无外部规则合成, 无预定义目标函数**  

---

## 二、指标定义

| 概念 | 公式 | 说明 |
|------|------|------|
| utilization | \|frames\| / memory_cap | 槽位占用率 |
| efficiency | 1 - Σ\|\|f.w - avg\|\| / (\|frames\|·avg) | 权重集中度 |
| stress | utilization · (1 - efficiency) | 内生生存张力, ∈[0,1] |
| 涌现 | 模式帧 weight ≥ median(所有帧 weight) | 模式在归纳中不会被淘汰, evaluate 返回 S2 |
| 编辑距离 | 1 - Levenshtein(sig₁, sig₂) / max(\|parts\|) | 签名匹配度 |
| S4 占比 | Σ(f.weight for "fn"/"sub" sig) / total_weight | 自指对记忆结构的影响 |
| 生存阈值 | median(所有帧 weight) | 由系统历史数据内生计算 |

## 三、实验结论

### 3.1 系统能从混杂信号中发现模式

| 条件 | 涌现率 | 说明 |
|------|--------|------|
| 噪声+事实+交换实例 (S4 催化剂) | ~35% | 系统发现交换结构 |
| 无 S4 | ~17% | S2 单独可涌现, 但概率低 |
| 噪声大幅升高 | ~5% | S4 在噪声下更重要 |
| 无 Q7 + S4 | ~35% | S4 可补偿缺失公理 |

### 3.2 涌现是阈值通过, 而非假阳性

- 0% 假阳性 (100 轮测试, 0 次噪声帧误判为 B2)
- 涌现有真实模式帧通过 median 权重阈值

### 3.3 零样本泛化 — 学到概念而非记忆数据

```
训练: 数字 1-8 的交换实例
测试: 50+100=100+50   → S2 PASS  (未见过的数字)
      x+5=5+x         → S2 PASS  (含变量)
      z+1=1+z         → S2 PASS  (陌生变量名)
```

### 3.4 消融锁定因果链

| 消融 | 涌现率 | p 值 |
|------|--------|------|
| 控制 (S4=1) | 35% | — |
| 无 S4 | 5% | 0.017 * |
| 冻结权重 | 0% | 0.003 ** |
| 随机阈值 | 0% | 0.003 ** |

**效应量**: S4 将涌现率从 17% 提升至 35% (约 2x)。S4 ≠ 必要条件, 是强力催化剂。

## 四、与主流研究的比较

### 4.1 自由能原理 (Friston et al., 2023-2024)

```
核心: 系统最小化"自由能"(预测误差)
GEME: 记忆管理最小化"应力"(槽位压力 × 权重集中度)

差异: 
  Friston: 预测误差 = 外部世界的拟合度
  GEME:    应力 = 内部编码的经济性
  
  自由能是观测者-环境互动的度量
  GEME 应力是系统自身编码容量的度量
```

### 4.2 基于用法的语法归纳 (MIT COLI, 2024)

```
核心: 从最小认知架构 + 序列记忆中归纳语法
GEME: 从记忆权重 + 签名匹配中归纳模式

差异:
  语法归纳: 统计序列 n-gram → 语法规则
  GEME:    权重竞争 + 编辑距离 → 模式生存
  
  语法归纳需要统计批次和历史序列
  GEME 只需要实时记忆管理
```

### 4.3 神经符号系统 (DeepMind/IBM)

```
核心: 神经网络学习 + 符号规则推理
GEME: 纯符号系统, 无神经网络

差异:
  神经符号: 梯度下降训练权重, 符号层做推理
  GEME:    无梯度, 权重=实例合并计数
  
  神经符号可解复杂问题, 但需要大量训练
  GEME 只需要实例流, 零样本泛化
```

### 4.4 主动推理 / Active Inference

```
核心: 系统通过行动减少预测误差
GEME: 系统通过归纳 (清理记忆) 减少应力

差异:
  主动推理: 行动改变外部世界 → 降低预测误差
  GEME:    归纳改变内部记忆 → 降低应力
  
  两者都是"系统自我调节"的实例
  GEME 是主动推理在符号空间的特例
```

### 4.5 GEME 的独特贡献

```
1. 零泛化: 不是 n-gram 匹配, 是结构签名匹配
2. 无梯度: 不依赖反向传播, 纯内存操作
3. 内生阈值: median 权重 = 涌现阈值, 非硬编码
4. S4 效应: 首次工程验证自指信号可辅助模式发现
5. 噪声弹性: 21k fps, 噪声比高达 3:1 时仍可涌现
```

---

## 五、局限性

```
1. S4 不是必要条件: 17% 无 S4 可涌现, S4 是催化剂非开关
2. 签名是预编码: 系统不发明新类型, 只匹配已有类型
3. InductonRule 已移除, 但 Q1-Q7 公理仍需预载
4. 递归签名溢出: 嵌套公式超出 compute_signature 深度
```
