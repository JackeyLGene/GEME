# GEME — 几何阶段 (GO) 启动报告

**日期**: 2026-05-03  
**前阶段**: PA (形式算术 7 条公理 → 涌现交换律)  
**本阶段**: GO (几何 1 条公理 → 涌现三角形全等条件)  
**核心理念**: `SAS → ASA → SSS` 链式涌现

---

## 一、实验设计 (三阶段递进)

### GO-1: 三角形全等自推演

```
公理: SAS (边角边全等)
  两条边及其夹角相等的三角形全等

涌现: ASA (角边角全等)
  两个角及其夹边相等的三角形全等
  
涌现: SSS (边边边全等)
  三条边分别相等的三角形全等

证明:
  SAS → ASA: 角交换 + 三角形内角和不变
  SAS → SSS: 边推导 + 余弦定理 (欧式第 5 公理介入)

与 PA 对比:
  PA: 7 公理 → 1 模式 (add_comm)
  GO: 1 公理 → 2 模式 (ASA, SSS)
  
  PA 的公理密度大, 涌现相对容易
  GO 只有 1 条公理, 系统需做更多推理工作
  如果 GO 成功, 验证涌现的普适性 (不依赖公理数量)
```

### GO-2: 欧式定理链式证明

```
起点: 欧式 5 公理 + GO-1 的全等条件
目标: 链式涌现平行线定理 → 内角和 → 勾股

Step 1: 平行公理 → 同位角相等 (引理)
Step 2: 同位角相等 → 内角和 180° (定理)
Step 3: 内角和 + 全等条件 → 勾股定理 (经典定理)

ProofChain 组件:
  - steps: [(rule, input_frames, output_frame, weight)]
  - 链式检测: frame_A.signature ∩ frame_B.signature ≠ ∅
  - 合并: 两帧共享关键词 → 高层"定理帧"
  
三角恒等式 (sin, cos):
  In: 勾股定理的证明过程中自然产生
  Out: 不做独立实验
```

### GO-3: 碰数式几何定理发现

```
目标: 无预设定理, 仅随机几何实例输入
系统自发: 高频等价类 → "候选定理" → 自行证明

"碰数" = 随机几何实例的高频碰撞
  → "三角形三条中线交于一点" 被反复观察到
  → 系统识别为模式 (weight ≥ median)
  → 系统: "这个结构经常出现, 应是定理"
  → 反向搜索证明路径

风险: 高 (全球无先例)
```
