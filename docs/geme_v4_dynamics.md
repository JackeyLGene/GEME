# GEME V4 动力学 — 向量预测 + 压缩压力

## 1. 单帧流程

```
input:  "1+2=2+1"
   │
   ▼
  W-Table → 向量: [0.67, 0.33, 0, 0, 0, 0.23]
                  num  plus  eq  var  forall  inst
   │
   ├── 历史模型(贝叶斯): v_hist = mean(window)
   │     └── gap_hist = |v - v_hist|
   │                    0.00 → 预测精准
   │
   ├── 维度模型(聚类): v_clust = nearest(v)
   │     └── gap_clust = |v - v_clust|
   │                    0.00 → 类内一致
   │
   ▼
  gap = 0.00  →  压缩率 = 1 - gap = 1.0
  freq = 8    →  压力 = 1.0 * min(1, 8/15) = 0.53
   │
   ▼
  应力累积 += 0.53  →  stress = 2.38
   │
   跃迁!
```

## 2. 核心反馈环

```
输入 F → W-Table 向量化 → v
  │
  ├── 预测: v_pred = bayes(v|history) + cluster(v)
  │
  └── gap = |v - v_pred|
  │     │
  │     ├ gap ≈ 0 → 压缩压力 = (1-gap) × freq
  │     │              高频 → 高压 → stress ↑  → 跃迁
  │     │
  │     └ gap ≫ 0 → 误差压力 = gap × freq
  │                  新的杂乱信号 → 低应力 → 积累慢
  │
  └── stress = 累积(压缩压力 + 误差压力)
```

## 3. 跃迁动力学

```
stress > 阈值 → 跃迁
  ├── L(E) + 1
  ├── 合成归纳规则 (PatternAnalyzer)
  ├── 注入 classifier → 重评估 ∀ 公式
  └── 涌现: ∀ 公式从 S3(不可判定) → S2(可推导)
```

## 5. 与之前的区别

```
V1-V3 (错误):
  x = 对象坐标  (预设符号位置)
  y = 自指坐标  (硬编码映射)
  stress = |y-x|  (固定间隙)
  → 分类器，非动力学

V4 (正确):
  v = 向量编码  (从数据计算)
  v_pred = 贝叶斯 + 聚类 (从历史学习)
  gap = |v - v_pred|  (学习偏差)
  pressure = (1-gap) × freq (压缩压力驱动)
  → 系统从数据中自学习模式
  → 高频模式产生压缩压力
  → 压力累积触发维度跃迁
```
