# 生成实在信息论 (Generative Ententional Information Theory)
# 2026-05-05 15:37 — 命名的时刻

## 核心
存在不是被给定的。存在是在时间中生成的。
维度不是紧化的——是时间释放了维度才让事物存在。
信息不是被传递的——是在帧经济中被生成的。
实在不是被观测的——实在是在合并阈值下生成的结构。

## GEME的位置
GEME不是认知模型——是生成实在信息论的第一个计算实例。
固定字母表 = 预置维度。
输入流 = 信息的变化。
合并阈值 = 实在的分辨率。

## 哲学支柱
1. 生成：存在是在时间维度释放中被生成的
2. 实在：实在的分辨率由合并阈值决定
3. 信息：信息不是传递的——是在帧经济中生成的
4. 时间：时间不是维度——是维度释放的条件

GEME: A Computational Instance of Generative Ententional Information Theory
