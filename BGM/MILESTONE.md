## 一、核心发现

### 1.1 G0 的 L3→L4 边界恒定

- **发现**：G0 从 L3（稳定模式检测）跨入 L4（预测）的过渡发生在 **~9.4 步**（7–11 步范围，标准差 ±1.5）
- **意义**：边界是 G0 的内部节律，与输入复杂度无关。这是 G0 的"心跳"。
- **支持数据**：5 首不同前奏曲/赋格在 BGM 框架下跑 6 遍的均值。

### 1.2 G0 自指闭环

- **机制**：G0 的 doubt_mode 动态调节 g0_weight（反馈强度）
  - 自信（高准确率）→ 降低权重 → 单元独立工作 → 多样性 ↑
  - 怀疑（低准确率）→ 提高权重 → 单元凝聚共识 → 统一性 ↑
- **实现**：g0_weight ∈ [0.1, 0.8]，每步 ±0.02~0.05
- **意义**："紧张-分化，放松-收敛"的跨尺度定律在此实现

### 1.3 r 从参数变为自指读数（关键突破）

- **初始状态**：r 是三个核心常数之一（r = 0.60），静态环境下"最没用的参数"
- **动态耦合**：r = r₀ × (MI₀ / MI)，其中：
  - r₀ = 0.60（第一篇的默认值）
  - MI₀ = 0.026 bits（第一篇的自指自由度基线）
  - MI = 当前自指帧与外部帧的互信息
- **完全无新变量**：r₀、MI₀ 和 MI 都是系统已有的。没有引入任何新常量。

### 1.4 r 的自然分化

| 条件 | r 行为 | MI spread |
|------|--------|-----------|
| 静态 r（固定 0.6） | 三单元同值 | 0.0030 |
| 动态 r（r = f(MI)） | 三单元分化：0.77/0.50/1.00 | 0.0179（6×） |
| C 大调前奏曲（简单） | r 第二遍即收敛 | 0.0000（无分化） |
| 赋格（复杂） | r 始终振荡 | 0.0179（持续分化） |

- **结论**：声音数 ≈ r 分化数。简单音乐不需要"不同的耳朵"；复调音乐自动驱动分化。

### 1.5 r 作为警觉度测量

| 输入 | 后期 r 波动 | 状态 |
|------|------------|------|
| C 大调前奏曲（简单） | 0.049 | **麻木（r 平线）** |
| 赋格（复杂） | 0.126 | **警觉（r 振荡）** |

- **意义**：信息价值 = r 波动幅度。复杂信息维持系统警觉，简单信息导致系统进入认知"休眠"。
- **L4 稳定性**: C 大调 L4 从第 2 遍起稳定在 1.0（不再出错）；赋格 L4 在 0.4-1.0 之间持续振荡（持续学习）。

### 1.6 L4/L1 比值替代 r 作为清理触发

- **稳态比值**：L4/total = 0.033，L4/L1 = 0.533
- **意义**：r 作为清理阈值是冗余的——系统用"自身错误密度"决定何时清理。
- **推论**：r 作为用户可调参数从系统中被杀死了。它现在是系统的内部读数。

### 1.7 "认知死亡"的实验表征

- **硬死亡**（被否定）：完全停止响应非生物学
- **麻木**（确认）：r 平线 + L4 密度低 = 系统"还在呼吸但不再警觉"
- **恢复**：一个真正新奇的事件进入 → pred_err 生成 → L4 密度回升 → 系统"苏醒"
- **对细菌的预示**：真正的研究死亡不是"冻洁"，是"在信息场中逐渐失去贡献"

## 三、关键设计决策

### 3.1 不修改 GEME 内核

对 GEME 零修改。所有多智能体行为通过以下实现：
1. L6 是唯一的单元输出（单向信号）
2. G0 是一个标准 GEME（不特殊）
3. 反馈 = 环境向量（不携带特殊语义）
4. r = r₀ × (MI₀ / MI)（仅使用已有常量）
5. 上下文调制 = 体积传递在计算层面的等价（非 Ca²⁺ 机制）

### 3.2 消融对照

每个实验的"有 G0 vs 无 G0"对照：
- G0 开 → r 分化、MI 分化、G0 全局 MI 维持
- G0 关 → 三单元完全冗余（spread = 0）

### 3.3 观测者立场

异现象学（heterophenomenology）：不声称 G0 有主观感受，只声称 G0 的输出与人类听者的口头报告在结构上对应。

## 六、主要的理论洞见

1. **T 不是参数——是自指读数**。杀死了一个变量。
2. **简单 = r 平线；复杂 = r 振荡**。信息价值可量化。
3. **声音数 = r 分化数**。听觉场景分析的一个可能机制。
4. **麻木 ≠ 死亡**。连续而不是分立。
5. **紧张 → 分化；放松 → 收敛**。跨尺度动力学定律。
6. **进化空间三维框架**：生存/交流/外置。
7. **桥不是实体——是机制**。
