# BGM 实验设计

> 两个实验、一个框架、一座桥。

## 核心设计原则

**不修改 GEME 核心**。GEME 始终从"环境"获取输入——无论是 MIDI 音符、营养梯度、还是邻居的信号，对 GEME 来说都是"外部输入向量"。

**两个实验共享同一套通信协议**。GEME 之间通过向量交换信息。一个 GEME 的输出（帧经济加权质心）成为另一个 GEME 的环境输入。

**巴赫有显式 G0，细度没有——这才是关键对比**。

- 巴赫实验有一个实体 G0 单元，接收三个平行 GEME 的输出，反馈一个"统摄信号"给它们。这便于读者理解：*有一个东西在听所有人说话*。
- 细度实验没有 G0 单元。16 个 GEME 纯粹通过局部邻居通信自组织。如果集体决策仍然涌现——说明 G0 不是必需的，而是机制的自然后果。

**巴赫教读者看见 G0；细度证明 G0 不需要被设计**。

**实验 3 是桥**。将细度群体涌现的节律施加给巴赫的 GEME，或者反过来。如果同一机制对两个极端系统产生同构响应，则验证了跨尺度普适性。

## 通信协议（两个实验共享）

```
每个 GEME 在每步 t 接收两类输入：

  外部环境:
    巴赫: MIDI 音符向量（当前音高 + 时值）
    细度: 营养源梯度向量 + 局部浓度

  邻居信号:
    巴赫: 其他 2 个 GEME 的输出向量（加权质心）
    细度: 4 个邻格 GEME 的输出向量

  输入融合:
    环境向量 × α + 邻居向量均值 × (1-α), α ∈ [0,1]
```

## 实验一：巴赫——3 个 GEME + 实体 G0 统摄

3 个标准 GEME 接收**完全相同的 MIDI 输入**。不堆叠——平行。

此外，存在一个**显式的 G0 单元**（同样是一个标准 GEME）。它的输入是三个平行 GEME 的输出向量的聚合。G0 的输出（其 L6 统摄信号）被反馈回三个平行 GEME，作为它们的外部环境输入之一。

```
MIDI 输入 ──► GEME_A ──┐
             ├► GEME_B ──┼──► G0（显式统摄单元）──► 反馈给 A/B/C
             └► GEME_C ──┘
```

预期：同一个输入，由于随机种子和合并路径的分叉，三个 GEME 会演化出不同的"听觉偏好"。一个可能对旋律敏感，一个对和声敏感，一个对节奏敏感。G0 在它们之间来回协调——像复调音乐的指挥。

**测量**：
- 三个 GEME 的 L6 输出信号是否随时间趋向同步？
- 它们对同一段音乐产生的预测误差模式是否互补？
- 移除 G0 反馈后，三个 GEME 的预测准确率是否下降（消融实验）？

## 实验二：细度——16 个 GEME 网格扩散，无 G0

16 个标准 GEME 排列在 4×4 网格中。**没有显式的 G0 单元**——唯一的通信渠道是邻居之间的局部向量交换。

每个 GEME 从两个来源获取信息：

1. **营养源梯度**：环境中有一个营养浓度场。每个 GEME 读取其位置的浓度值
2. **邻居信号**：4 个邻格（东南西北）的 GEME 输出向量（帧经济的加权质心）

参考：铜绿假单胞菌（Pseudomonas aeruginosa）——群体感应（quorum sensing）的模式生物。

**预期**：没有中央控制器，没有显式 G0——只有局部通信。但 16 个 GEME 的帧经济会自然进入同步（类似细度群体的信息素同步），产生集体决策——"向营养源移动"。如果无 G0 也能产生集体行为，则 G0 是机制的自然后果，而非预设。

**测量**：
- 群体定向准确率（向高营养方向移动的成功率）
- 相位收敛速度（多少步后进入同步）
- 与巴赫实验的收敛曲线对比（KS 检验）——即使一个有 G0、一个没有

## 实验三：桥——施加坡赫节律于细度

将细度实验中的"通信节律"（信息素浓度的时序涨落模式）作为时序信号，施加给巴赫实验中的 GEME。

如果 G0 层只对"时序复杂度"敏感、不对"含义"敏感——那么巴赫听到的是音乐，细度收到的是信号——G0 应产生相同的统计同构响应。

**核心主张**：G0 不是"音乐的欣赏者"也不是"细度的语言"——它是时序自指的能力本身。

## 测量指标框架（两个实验统一）

| 指标 | 定义 | 巴赫 | 细度 |
|------|------|------|------|
| 相位收敛度 | 各GEME帧更新周期的标准差 | 三个L6的同步 | 16个L6的同步 |
| 全局有序熵 | 预测准确率的分布熵 | 高→同步 | 低→同步 |
| 信息处理上限 | 去掉G0通信后是否崩溃 | 3→1 | 16→1 |
| G0 拓扑 | 邻居信号的网络结构 | 全连接 | 网格扩散 |
