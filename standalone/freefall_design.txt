# 自由落体的GEME实验设计

## 物理对应
自由落体：物体在质量场中沿测地线运动。
GEME对应：一个"探针"输入穿过密度梯度场。

## 实验设置
1. 一个静态质量分布在原点（高密度区）
2. 一个探针信号从远处（x=100）开始，逐步移向原点（x=0）
3. 探针每秒发出一个"滴答"信号
4. 测量每个位置处滴答的帧生成率

## 预测
- 远处（低密度）：滴答被合并→帧少→"时间快"
- 近处（高密度）：滴答被分散→帧多→"时间慢"
- 帧生成率的曲线 = dτ/dt = √(1 - rs/r) 的形状

## 实现
```python
# 探针在密度场中自由下落
for x in range(100, 0, -1):
    dens = M / (1 + 0.1 * x)  # 当前位置的密度
    发送滴答信号到GEME
    记录当前位置生成的帧数
```

数据点和GR理论曲线做拟合。
